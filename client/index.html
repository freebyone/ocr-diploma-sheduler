Конвертер PDF в изображения

Загрузите PDF файлы для обработки. Python API разобьёт каждый файл на отдельные страницы, преобразует их в изображения, загрузит в MinIO и отправит пути к файлам в Kafka.

Выберите PDF файлы для обработки

Максимальный размер файла: 50 MB

Поддерживаются только PDF файлы

Настройки обработки

Как работает обработка:

  • 1. Загрузка PDF: Вы выбираете PDF файлы через форму
  • 2. Отправка в Python API: Файлы отправляются на сервер для обработки
  • 3. Анализ PDF: Python API проверяет количество страниц в каждом файле
  • 4. Конвертация: Каждая страница конвертируется в отдельное изображение
    Пример: pdf1 → img1.1.jpg, img1.2.jpg; pdf2 → img2.1.jpg, img2.2.jpg
  • 5. Сохранение в MinIO: Все изображения загружаются в MinIO хранилище
  • 6. Уведомление через Kafka: Пути к сохраненным файлам отправляются в Kafka
