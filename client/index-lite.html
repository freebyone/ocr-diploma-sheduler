Загрузка PDF и Excel файлов

Выберите тип файлов для загрузки: PDF для сканированных документов или Excel для учебных планов. Excel файлы загружаются в MinIO хранилище для последующей обработки.

Формат имени файла: имя должно начинаться с числового префикса, например 0001_Scan_приложение.pdf. Первая страница PDF будет сохранена в MinIO как 0001.jpg.
